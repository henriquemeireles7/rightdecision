## From founder

### Course duration and pacing
- Total: ~24 hours
- 3 acts across 3 months
- 2 hours per week
- One act per month

### Class structure
- Theory classes: teach the concept
- Practical class (last class of each module): explains how to fill up the practice document
- Practice has 3 assets:
  1. The class itself (in the platform, as usual)
  2. A questionnaire (docx file with structured questions for manual work)
  3. An AI prompt (in the last page of the docx, used AFTER filling the questionnaire manually)

### The "doing the work" flow
1. Student watches the practical class
2. Downloads the docx
3. Fills the questionnaire manually (this IS the manual-first work)
4. Pastes the AI prompt from the last page into ChatGPT/Claude
5. Uploads or pastes their filled questionnaire
6. AI asks follow-up questions to refine
7. AI outputs the final document for that phase
8. Student saves all documents into an AI project (ChatGPT/Claude project folder)

### Exercise design: manual-first, AI-second confirmed
Manual questionnaire first. AI refines second. The docx IS the exercise.
