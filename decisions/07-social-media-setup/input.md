## Core Strategy: AI-First Distribution

The fundamental insight is that this is NOT a "creators making content" strategy. It's an **AI-first content production system** where:

1. **Brand accounts (rightdecision)** = 100% AI-generated from day 1. Content is tightly mapped to methodology: individual lessons, acts, mini-VSLs about specific concepts or angles. Faceless content + content using Henry and Indy's existing footage/faces.

2. **Personal accounts (indykaz, henrykaz)** = start with real recordings (podcasts, manual posts), but the roadmap is to develop AI voice clones, AI character versions, so everything evolves to AI-first. Manual posting remains an optional override.

3. **Future dark channels** = 100% AI-generated, faceless, for specific niches. No connection to personal brands.

The division between accounts is not by content topic — it's by **process** (AI-generated vs human-recorded) and **ICP** (Indy speaks to women about self-development, Henry speaks to men about business/making money online, brand speaks to everyone about the methodology).
