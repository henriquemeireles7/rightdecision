# The Right Decision

A yearly platform ($197/year) that teaches you the one decision that actually matters right now, then uses AI to decompose it into commitments you can execute daily.

## Stack

Bun · Hono · Preact · Drizzle · Zod · PostgreSQL · Better Auth · Stripe · Tailwind

## Getting Started

```bash
bun install
bun run dev      # start dev server (hot reload)
bun run check    # lint + typecheck + test
bun test         # run tests only
bun run lint     # fix lint + format issues
```
